An intelligent web application that detects and analyzes fake news in real time using NLP-based classification models behind a clean, accessible interface.

| Attribute | Details |

| **Scale** | Trained on 40K+ labeled news articles |
| **Performance** | ~94% classification accuracy, sub-second inference |
| **Security** | Input sanitization, rate limiting, validated API contracts |
| **Impact** | Enables instant credibility checks for everyday readers |
| **Repository** | [Walid-Ysn/fake-news-detection](https://github.com/Walid-Ysn) |
